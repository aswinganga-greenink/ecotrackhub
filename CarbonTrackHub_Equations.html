Gram Panchayat Anjarakandi  |  GHG Protocol & IPCC Guidelines  |  Generated: March 2026

1. Emission Factors

All factors follow the GHG Protocol Corporate Standard, IPCC AR5 values, and India-specific CEA data.

1.1 Activity Emission Factors

Source Factor Unit Reference
🔌 Electricity (Grid) 0.716 kg CO₂e per kWh CEA India Grid Emission Factor 2023
⛽ Diesel 2.68 kg CO₂e per litre US EPA AP-42 / DEFRA 2023
🚗 Petrol 2.32 kg CO₂e per litre US EPA AP-42 / DEFRA 2023
🗑️ Solid Waste 0.586 kg CO₂e per kg IPCC 2006 – Average landfill CH₄ + CO₂
💧 Water 0.000344 kg CO₂e per litre UK Water Industry Research / DEFRA

1.2 Offset Factors

Source Factor Unit Reference
🌳 Tree (mature) 21.77 ÷ 12 kg CO₂ absorbed per tree per month US Forest Service average – 48 lbs/yr ≈ 21.77 kg/yr
☀️ Solar Energy 0.716 kg CO₂ offset per unit (kWh) Equals grid factor (displaced grid generation)

2. Core Equations

2.1 Monthly Total Emissions

Formula: E_total = (E_kwh × 0.716) + (D_liters × 2.68) + (P_liters × 2.32) + (W_kg × 0.586) + (Wtr_liters × 0.000344) Where: E_kwh = Electricity consumed (kWh) D_liters = Diesel consumed (litres) P_liters = Petrol consumed (litres) W_kg = Solid waste generated (kg) Wtr_liters = Water consumed (litres)

2.2 Monthly Total Offsets

Formula: O_total = (T_planted × 21.77 / 12) + (S_units × 0.716) Where: T_planted = Number of trees planted this month S_units = Solar energy generated (kWh / units) 21.77 = kg CO₂ absorbed per mature tree per year 12 = Months per year (monthly proration)

2.3 Net Carbon Footprint

Formula: Net = E_total − O_total Interpretation: Net ≤ 0 → ✅ Carbon Neutral (offsets exceed emissions) Net > 0 → ❌ Net Emitting (emissions exceed offsets)

2.4 Average Monthly Emissions (per user / per panchayat)

Formula: E_avg = (Σ E_total over N months) / N E_per_capita = E_total / Population

2.5 Sector-wise Emission Breakdown

Individual sector contributions: E_electricity = E_kwh × 0.716 E_diesel = D_liters × 2.68 E_petrol = P_liters × 2.32 E_waste = W_kg × 0.586 E_water = Wtr_liters × 0.000344 Percentage share of each sector: Share_sector (%) = (E_sector / E_total) × 100

4. AI Prediction Logic

The prediction engine uses Google Gemini AI. No manual formula is applied — the AI analyses the calculated_total_emission_kg series and forecasts 6 months forward.

📌 Forecast start = Month immediately after the last data entry (e.g. if last data is March 2025 → forecast starts April 2025).
📌 Results are cached in localStorage and only re-generated when new data is submitted.

5. Database – Clear Commands

⚠️ These commands permanently delete data. Make a backup first if needed.

5.1 Clear only monthly data (keeps users and settings)

# From the ecotrackhub project root: sqlite3 backend/carbontrackhub.db "DELETE FROM monthly_data;"

5.2 Clear monthly data AND carbon metrics

sqlite3 backend/carbontrackhub.db "DELETE FROM monthly_data; DELETE FROM carbon_metrics;"

5.3 Clear all non-admin users (keeps admin account)

sqlite3 backend/carbontrackhub.db "DELETE FROM monthly_data; DELETE FROM users WHERE role='user';"

5.4 Clear EVERYTHING – full reset (keeps table structure)

# ⚠ FULL WIPE – all data, all users except admin will be gone sqlite3 backend/carbontrackhub.db \ "DELETE FROM monthly_data; DELETE FROM carbon_metrics; DELETE FROM otp_verifications; DELETE FROM users WHERE role='user';"

5.5 View current data before clearing

# Count records per table: sqlite3 backend/carbontrackhub.db \ "SELECT 'monthly_data', COUNT(*) FROM monthly_data UNION ALL SELECT 'users', COUNT(*) FROM users UNION ALL SELECT 'carbon_metrics', COUNT(*) FROM carbon_metrics;" # Show all monthly data entries: sqlite3 backend/carbontrackhub.db \ "SELECT month, year, user_id FROM monthly_data ORDER BY year, month;"

5.6 Using Python (alternative – reads same DB path)

# Run from backend/ directory: cd backend source venve/bin/activate python - <<'EOF' from database import SessionLocal from models import MonthlyData db = SessionLocal() db.query(MonthlyData).delete() db.commit() db.close() print("✅ Monthly data cleared.") EOF
